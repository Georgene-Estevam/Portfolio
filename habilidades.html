HTML

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas na web. Ele estrutura o conteúdo que aparece em um site, como textos, imagens, vídeos, links e muito mais.

CSS

CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem usada para estilizar elementos escritos em uma linguagem de marcação como HTML. Ele separa o conteúdo da representação visual do site, permitindo que você altere a cor do texto e do fundo, fonte, espaçamento entre parágrafos, crie tabelas, ajuste imagens e muito mais.

JavaScript

JavaScript é uma linguagem de programação de alto nível, leve e interpretada, amplamente utilizada para criar páginas web dinâmicas e interativas. Ele permite adicionar funcionalidades como animações, validação de formulários, atualizações de conteúdo em tempo real e muito mais.

SQL

SQL significa “Structured Query Language”, ou “Linguagem de Consulta Estruturada”, em português. É uma linguagem de programação para lidar com banco de dados relacional (baseado em tabelas).

Git

Git é um sistema de controle de versão distribuído que permite que desenvolvedores colaborem em projetos de software de forma eficiente e organizada. Ele foi criado por Linus Torvalds em 2005 e se tornou uma ferramenta essencial no desenvolvimento moderno, permitindo que múltiplos programadores trabalhem simultaneamente em um mesmo projeto sem conflitos.

Sistema Operacional Linux

Criado por Linus Torvalds, Linux é um sistema operacional de código aberto, o que significa que é livre para uso e modificação por qualquer pessoa interessada. Desde sua criação em 1991 por Linus Torvalds, o Linux se tornou um dos sistemas operacionais mais populares e amplamente utilizados no mundo.
